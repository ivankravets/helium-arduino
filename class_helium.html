Detailed Description

Enables communication with the Helium Atom.

This class offers the basic ability to communicate with the Helium Atom module.

See the Basic.ino sketch for an example on how to create a channel and send to it.

int Helium::connect ( struct connection *  connection,
uint32_t  retries 
)

Connect the Helium Atom to the network.

This tries to connect the Helium Atom to a nearby Element and the Helium Network. Connections can be quick or slow. A quick connection attempt to re-connect to the network using revious connection state which was retrieved using sleep. If the quick connect was not requested or if it fails a normal, slower connection attempt is made.

Connect is an asynchronous operation. The given are a convenience to attempt a number of retries before returning the current connection state.

Parameters
connectionA previously retrieved connection structure that reflects sleep state.
retriesThe number of retries to attempt to connect.
Returns
helium_connect_status

int Helium::sleep ( struct connection *  connection)

Disconnects the Atom from the network and puts it in sleep mode.

This disconnects the Helium Atom from the network and puts it in sleep mode. If a connection structure is given, it is filled with sleep state which can be used in a sub-sequent connect call to quick connect to the network.
